# Lightblues Tap

## How do I install these formulae?

`brew install lightblues/tap/<formula>`

Or `brew tap lightblues/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lightblues/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
